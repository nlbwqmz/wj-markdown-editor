# 2.19.0

## 新增

- 新增文件管理栏搜索，支持按名称筛选当前目录中的文件和文件夹；搜索无结果时会直接给出空状态提示，定位内容更方便。
- 新增 PNG、JPEG 导出到剪贴板能力，导出菜单现在区分“导出到文件”和“导出到剪贴板”，复制图片到聊天或文档更直接。

## 优化

- 优化文档大纲面板的展示和交互，目录层级更清晰，长标题可悬停查看完整内容，点击目录后的跳转与当前标题高亮跟随更稳定。
- 优化编辑器、预览区等分栏拖拽线与滚动条的交互反馈，悬停和拖动时的视觉提示更明确，调整布局时更容易命中。

## 修复

- 修复点击编辑器外部工具栏后，格式化等操作可能导致编辑器状态不同步的问题，减少内容回跳或状态错乱。
- 修复暗黑主题下编辑器行号区域右侧边框丢失的问题，界面分区显示恢复正常。
- 修复打开不支持的文件类型时提示不准确的问题，错误信息现在更清晰。
