# 2.16.2

## 新增

- 新增打开 Markdown 文件失败时的系统通知提醒，遇到文件被占用、权限不足等场景时，能更直接获知失败原因；Windows 下通知标题与图标显示也更稳定。

## 优化

- 优化设置页配置修改体验，`recentMax` 输入会自动规范，取消目录选择不再污染草稿，配置保存失败时也会给出更明确的提示。
- 优化预览区代码块交互，工具栏改为右上角浮层展示，语言标记与复制操作更聚焦，配色也会随当前代码高亮主题自适应。
- 优化预览主题和代码块样式的一致性，不同预览主题下的代码块外观更统一，切换主题时视觉表现更稳定。

## 修复

- 修复部分异常配置值或历史配置内容可能导致设置加载、更新失败的问题，减少 `recentMax`、快捷键等配置项在边界场景下的异常。
- 修复部分预览主题在暗黑模式下表格、引用块和音频控件层次不清的问题，提升暗色场景下的可读性。
- 修复代码块工具栏在悬停、高亮异常或键盘操作场景下可能出现的显示不稳定问题。
