# 2.19.3

## 新增

- 文件管理栏新增排序菜单，支持按名称、类型、修改时间进行升序或降序查看，并保留当前排序偏好，目录浏览更灵活。
- 预览区新增对 HTML `<img>` 图片标签的资源管理支持，使用原生 HTML 插入的图片也能参与右键菜单、资源定位和引用删除操作。

## 优化

- 优化了文件管理栏交互，新建文件夹和新建 Markdown 入口合并为统一菜单，定位当前文档时也会自动滚动到目标条目，回到正在编辑的文件更直接。
- 优化了顶部栏操作入口，新增可点击的视图切换按钮，编辑页与预览页之间切换更方便。
- 优化了设置页的配置保存体验，主题、语言、快捷键等设置在修改、重置和失败回退时的状态同步更稳定，减少草稿丢失或界面状态不一致的问题。

## 修复

- 修复了 keep-alive 场景下预览区在失活状态仍可能触发 Mermaid 渲染的问题，切换视图或主题后返回页面时，预览内容刷新更稳定。
- 修复了文件管理栏按修改时间排序时目录刷新和特殊目录条目识别不准确的问题，目录变化后的排序结果更可靠。
