# 2.16.0

## 新增

- 新增编辑页与预览页的滚动锚点保持功能，在切换视图、重新激活窗口或恢复文档后，能更自然地回到之前浏览的位置。
- 新增预览区锚点链接平滑滚动能力，点击目录、脚注或文内锚点后，页面会更顺畅地定位到目标内容。

## 优化

- 优化预览搜索体验，支持在预览页、设置页、使用指南等视图统一搜索和高亮定位，跨视图切换时的关闭与清理行为更加一致。
- 优化文档打开、保存和外部变更处理链路，在连续保存、窗口切换、最近文件恢复以及 Windows 路径大小写差异等场景下，状态同步更稳定。
- 优化文档与本地资源定位体验，打开文档目录或资源所在位置时路径解析更可靠，文档未保存或路径无效时也会给出更明确的提示。

## 修复

- 修复导出图片时可能出现空白帧或界面阻塞的问题，导出过程更加稳定。
- 修复关闭窗口时最近文件列表可能更新异常的问题。
- 修复文档快照恢复、关闭确认和外部修改应用过程中，可能出现的内容不同步、提示重复或旧状态残留问题。
