# 2.18.0

## 新增

- 新增文件管理栏，可直接选择目录、返回上一级、定位到当前文件所在目录，并支持新建文件夹、新建 Markdown 文件以及在当前窗口或新窗口打开文件。
- 新增主窗口原生全屏功能，支持通过菜单或快捷键快速切换，更适合专注写作和演示场景。

## 优化

- 优化预览区资源右键菜单，补齐复制图片、复制图片链接或资源链接、复制 Markdown 引用、图片另存为等操作，资源管理流程更完整。
- 优化文件管理栏的图标识别、路径显示、空状态与当前文件高亮表现，并支持默认显示和快捷键切换，目录浏览与文件切换更顺手。

## 修复

- 修复预览区资源在 Windows 路径、带查询参数或片段、被外层链接包裹等场景下识别不准或操作异常的问题，降低误判和误操作风险。
- 修复部分远程资源处理与保存失败提示不准确的问题，提升远程图片复制、另存为和删除操作的稳定性与安全性。
- 修复文件管理栏目录刷新、目录监听和新建 Markdown 文件时的异常场景，避免面板静默空白或打开失败。
- 修复拖拽调整分栏后窗口缩放可能导致布局冻结的问题，提升编辑页布局稳定性。
